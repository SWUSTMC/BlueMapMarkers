# BlueMapMarkers 
### 玩家指令:  
/bmm help 查看帮助  
/bmgui 打开标记点GUI  
/bmm add name icon 添加一个标记点  
这里name和icon为可选参数  
若包含name和icon则为持久标记  
不包含则为临时标记(服务器重启不会加载)  
/bmm icon 查看icon帮助(可在config.yml中修改)
### 管理员指令
权限节点：bluemapmarkers.admin  
/bmma help 查看管理员帮助  
/bmma reload 插件重载  
### 注意事项
icon参数必须是可以访问的图片直链  
GUI中的删除无视是谁创建  
该插件需要以[BlueMap](https://github.com/BlueMap-Minecraft/BlueMap)作为前置
